## ğŸ’¡ Proposed Solution

The proposed system is a **low-cost assistive care solution** that combines **hardware interfaces** with **software-based IoT communication**.

Patients can communicate their needs using:

* Keypad for simple input
* Voice-assisted interaction for ease of use

### Caregiver Side (Software)

* Real-time notifications via an **IoT dashboard**
* SMS alerts for immediate attention

This ensures faster intervention, better monitoring, and improved patient safety.
